## QA Study Notes - 2026-06-22

### ISTQB CTAL-TAE - Chapter 3 (gTAA)
- gTAA = Generic Test Automation Architecture
- Layers: Generation, Definition, Execution, Adaptation
- SUT (System Under Test) communicates via Test Adaptation Layer
- TAF (Test Automation Framework) should be independent of SUT

### Key Concepts
- Test automation should not replace exploratory testing
- Maintainability is a core quality attribute of a TAF
